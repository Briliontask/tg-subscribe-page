# TG subscribe page

Одностраничный HTML-шаблон с одной кнопкой по центру (`SUSCRIBIRSE`).

## Как это теперь работает

- В `admin` вы вставляете CRM-ссылку и получаете короткий URL `/?id=...`.
- Сопоставление `id -> URL` сохраняется через Netlify Function в Netlify Blobs.
- Новые ссылки создаются без правки файлов и без перезаливки сайта.

## Условная админка

Есть страница `https://your-domain.com/admin/`.

Что умеет:

- вход по условному паролю;
- вставить ссылку из CRM;
- получить короткую ссылку вида `/?id=c17` сразу в рабочем виде.

### Важно про безопасность

Это быстрый вариант без бэкенда. Пароль хранится на фронтенде и не является
надежной защитой.

### Смена пароля

В Netlify Site settings -> Environment variables добавьте:

`ADMIN_PASSWORD=ваш_пароль`

Если переменная не задана, используется значение по умолчанию из кода:

`const ADMIN_PASSWORD = "change-me-123";`

## Важно для Netlify деплоя

Чтобы Netlify Functions работали, сайт нужно деплоить не простым drag-and-drop
статикой, а через Git (или Netlify CLI). Это разовая настройка.
